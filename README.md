## パッケージ分ける方法がスーパー特殊でマジ卍

1. GOPATHにプロジェクトのルートを設定
1. src/github/自分のアカウント名/アプリ名　のディレクトリを作成
1. 使いたいモジュール書く
1. importにgithubから始まるパッケージのパスを書く

これでモジュールを参照して利用できるようになる
